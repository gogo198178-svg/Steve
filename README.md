# 林國華個人網站

這是一個可直接發布的靜態個人網站。

## 檔案

- `index.html`：網站主頁
- `research.html`：個人著作與研究清單頁
- `styles.css`：網站樣式
- `profile.jpg`：個人照，用於首頁左側個人區塊

## 發布方式

可將本資料夾內容上傳至下列任一平台：

- GitHub Pages
- Netlify
- Vercel
- 一般網站主機或學校伺服器

發布時請讓 `index.html` 位於網站根目錄。若平台要求指定發布資料夾，選擇本資料夾即可。
